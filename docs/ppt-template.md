# PPT设计建议

## 颜色方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色 | 渐变紫蓝 | #667eea → #764ba2 |
| 辅助色 | 浅灰 | #f5f5f5 |
| 强调色 | 橙色 | #ff6b6b |
| 文字色 | 深灰 | #333333 |

## 字体建议

| 用途 | 字体 |
|------|------|
| 标题 | 思源黑体 Bold / 阿里巴巴普惠体 |
| 正文 | 思源黑体 Regular |
| 英文 | Roboto / Inter |

## 设计原则

1. **一页一个重点**：不要塞太多内容
2. **多用图表**：表格、流程图、对比图
3. **保持统一**：颜色、字体、版式统一
4. **适当留白**：不要堆满文字
