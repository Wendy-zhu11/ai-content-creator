# AI培训从业者知识体系

## 一、你需要掌握的核心知识

### 1.1 AI工具知识（必会）

| 类别 | 工具 | 学习优先级 |
|------|------|------------|
| **对话AI** | ChatGPT、Claude、DeepSeek、文心一言 | 🔴 必会 |
| **AI生图** | Midjourney、Stable Diffusion、通义万相 | 🔴 必会 |
| **AI写作** | Notion AI、Jasper、Copy.ai | 🟡 重要 |
| **AI办公** | Office Copilot、飞书AI、钉钉AI | 🟡 重要 |
| **AI视频** | Runway、Pika、Sora | 🟢 了解 |
| **AI音频** | 剪映AI、讯飞星火、ElevenLabs | 🟢 了解 |

### 1.2 应用场景知识（必会）

| 场景 | AI能做什么 | 学习内容 |
|------|------------|----------|
| **内容创作** | 文案、图片、视频生成 | 小红书、抖音、公众号内容生产 |
| **办公提效** | PPT、Excel、文档自动化 | 日常办公场景AI化 |
| **营销获客** | 广告文案、社媒运营 | AI营销全流程 |
| **客服销售** | AI客服、销售话术 | 企业降本提效 |
| **数据分析** | AI分析报告、数据可视化 | 决策辅助 |

### 1.3 培训方法论（必会）

| 模块 | 内容 | 学习方式 |
|------|------|----------|
| **课程设计** | 如何设计一门AI课程 | 参考优质课程 |
| **教学方法** | 如何让学员学会 | 实战教学法 |
| **落地陪跑** | 如何帮企业落地 | 项目管理+咨询 |
| **案例包装** | 如何展示成果 | 内容营销 |

## 二、学习路径（2周速成版）

### Week 1：AI工具精通

| 天 | 学习内容 | 实操任务 |
|----|----------|----------|
| Day 1 | ChatGPT/Claude基础与高级技巧 | 用AI写10篇不同类型文案 |
| Day 2 | DeepSeek/文心一言对比使用 | 国产AI工具测评报告 |
| Day 3 | AI生图（Midjourney/通义万相） | 生成10张不同风格图片 |
| Day 4 | AI办公工具（PPT/Excel） | 用AI完成一份完整PPT |
| Day 5 | AI视频/音频工具 | 制作一条AI短视频 |
| Day 6 | 工具整合与自动化 | 搭建一个AI工作流 |
| Day 7 | 复盘+输出 | 写一篇AI工具使用心得 |

### Week 2：培训能力构建

| 天 | 学习内容 | 实操任务 |
|----|----------|----------|
| Day 1 | 研究市面AI课程（5-10门） | 分析课程结构、优缺点 |
| Day 2 | 设计自己的AI课程大纲 | 产出课程框架 |
| Day 3 | 开发第一节课程内容 | 写逐字稿+PPT |
| Day 4 | 试讲+录制 | 录制课程视频 |
| Day 5 | 修改优化 | 根据试讲调整 |
| Day 6 | 包装课程+定价策略 | 完成课程上架准备 |
| Day 7 | 发布+获客测试 | 小红书/朋友圈推广 |

### 4.2 课程开发模板

```
课程结构（标准版）：

1. 认知篇：为什么需要AI？（痛点+案例）
2. 工具篇：AI工具有哪些？（分类介绍）
3. 操作篇：AI怎么用？（手把手教学）
4. 场景篇：AI能解决什么问题？（实战案例）
5. 落地篇：如何持续用AI？（方法论+工具包）
```
