## 模板评分体系

### 评分公式

```
总分 = 爆款指数(40%) + 时效性(20%) + 使用率(25%) + 满意度(15%)
```

### 各维度说明

| 维度 | 计算方式 | 权重 |
|------|----------|------|
| 爆款指数 | (点赞+收藏*2+评论*3) / 曝光量 | 40% |
| 时效性 | 1 - (当前日期-发布日期)/90 | 20% |
| 使用率 | 用户选择次数 / 总生成次数 | 25% |
| 满意度 | 正面反馈数 / 总反馈数 | 15% |

### 淘汰机制

| 条件 | 操作 |
|------|------|
| 总分 < 0.5 | 标记为「备选」 |
| 连续4周无使用 | 归档 |
| 过期时间已到 | 自动休眠 |
