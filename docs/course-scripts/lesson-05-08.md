### 【第一部分：Midjourney快速上手】（约10分钟）

#### 什么是Midjourney？

Midjourney是目前最强大的AI绘画工具，能生成商业级的图片。

**优势**：
- 画质高，细节丰富
- 风格多样
- 商业可用

**劣势**：
- 需要付费（$10/月起）
- 需要Discord账号
- 需要翻墙

#### 如何使用Midjourney

**步骤一：注册Discord**
1. 访问discord.com
2. 注册账号
3. 下载客户端或用网页版

**步骤二：加入Midjourney**
1. 访问midjourney.com
2. 点击"Join the Beta"
3. 加入Discord服务器

**步骤三：生成图片**
1. 进入新手频道（newbies）
2. 输入 `/imagine`
3. 输入提示词
4. 等待生成

#### 提示词公式

**公式**：主体 + 风格 + 细节 + 参数

**示例**：
```
A beautiful girl, anime style, soft lighting, detailed --ar 3:4
一个美丽的女孩，动漫风格，柔和光线，细节丰富，竖版比例
```

**常用参数**：
- `--ar 3:4` 竖版比例（小红书）
- `--ar 16:9` 横版比例（视频封面）
- `--v 6` 使用第6版模型
- `--s 100` 风格化程度

#### 实战演示

**演示一：生成小红书配图**
```
/imagine prompt: A cozy coffee shop interior, warm lighting, 
aesthetic, soft colors, instagram style --ar 3:4 --v 6

结果：温馨的咖啡店内饰，适合小红书探店配图
```

**演示二：生成品牌Logo**
```
/imagine prompt: Minimalist logo for a coffee brand, 
simple line art, clean design, vector style --ar 1:1

结果：简约咖啡品牌Logo
```

### 【第二部分：小红书壁纸号运营】（约8分钟）

#### 为什么选壁纸号？

- 需求大（人人都要壁纸）
- 内容简单（一张图+简短文字）
- 变现快（接广告、带货、引流）

#### 运营步骤

**第一步：账号定位**
- 选择细分领域：唯美、二次元、风景、文字壁纸
- 风格统一
- 人设清晰

**第二步：内容生产**
- 用Midjourney或通义万相批量生成
- 每天生产10-20张
- 选最好的3-5张发布

**第三步：内容发布**
- 时间：早8点、午12点、晚8点
- 标题：吸引眼球
- 话题：#壁纸 #手机壁纸 #高清壁纸

**第四步：互动涨粉**
- 积极回复评论
- 引导点赞收藏
- 建立粉丝群

**第五步：变现**
- 接广告（粉丝1万以上）
- 带货（壁纸相关的产品）
- 定制服务（引流到私域）
