## 【第一部分：ChatGPT账号注册与基础设置】（约5分钟）

### 1.1 如何注册ChatGPT

**方法一：官网注册（推荐）**

| 步骤 | 操作 |
|------|------|
| 1 | 打开官网：chat.openai.com |
| 2 | 点击 Sign up（注册） |
| 3 | 用邮箱或Google账号注册 |
| 4 | 完成手机验证（可能需要国外手机号） |
| 5 | 登录即可使用 |

**注意事项**：
- 国内直接访问可能需要网络工具
- 如果没有国外手机号，可以用接码平台（搜索"ChatGPT 接码"）

**方法二：使用第三方镜像站**

如果你不想折腾，可以用国内的镜像站：
- 搜索"ChatGPT 镜像站"
- 选择靠谱的平台
- 注意：镜像站可能不如官方稳定

### 1.2 免费版 vs 付费版如何选择

| 对比项 | 免费版（GPT-3.5） | 付费版（GPT-4） |
|--------|------------------|-----------------|
| 价格 | 免费 | $20/月（约145元） |
| 智能程度 | 够用 | 更聪明 |
| 回答质量 | 良好 | 优秀 |
| 速度 | 快 | 稍慢 |
| 推荐人群 | 新手 | 进阶用户 |

**我的建议**：
- 新手先用免费版，把基础功能摸透
- 有收入后升级付费版，提升输出质量
- 付费版不是必须的，免费版也能赚钱

### 1.3 基础设置优化

登录后，建议做以下设置：

**设置一：自定义指令**

路径：Settings → Personalization → Custom Instructions

填写：
```
你希望ChatGPT如何回应你：
你是一个专业的内容创作助手，擅长写小红书、公众号、商业文案等多种类型的内容。你的回答要简洁、实用、有创意。

你希望ChatGPT了解你什么：
我是一名AI副业从业者，主要用AI做内容创作和服务接单。我希望你帮我快速产出高质量内容。
```

**设置二：聊天历史**

- 如果不想让OpenAI用你的数据训练模型，可以关闭聊天历史
- 路径：Settings → Data Controls → Chat History & Training → 关闭

## 【第二部分：提示词工程——让AI输出高质量内容】（约12分钟）

接下来是最重要的部分：**提示词工程**。

很多人用AI效果不好，不是AI不行，是提示词写得不好。

### 2.1 什么是提示词？

提示词就是你给AI的指令。

**提示词质量决定输出质量。**

| 提示词 | AI输出 |
|--------|--------|
| 写一篇文章 | 随机、质量不稳定 |
| 写一篇关于XX的小红书笔记，风格活泼，带emoji | 精准、质量稳定 |

### 2.2 提示词万能公式

我给你一个万能公式，照着写就能出好结果：

**万能公式 = 角色 + 任务 + 要求 + 示例**

| 要素 | 说明 | 示例 |
|------|------|------|
| **角色** | 让AI扮演什么角色 | "你是一个小红书爆款文案专家" |
| **任务** | 让AI做什么 | "请帮我写一篇关于春日穿搭的笔记" |
| **要求** | 对输出的具体要求 | "300字左右，带emoji，风格活泼" |
| **示例** | 给AI看一个例子 | "参考这个风格：..." |

**完整示例**：

```
【角色】你是一个小红书爆款文案专家，擅长写吸引年轻女性的内容。

【任务】请帮我写一篇关于"春日穿搭"的小红书笔记。

【要求】
- 300字左右
- 标题要吸引眼球，带emoji
- 正文分3-5段，每段1-2句话
- 风格活泼亲切，像朋友分享
- 结尾引导互动
- 加3-5个话题标签

【示例风格参考】
标题：🌿春日穿搭公式！5套look承包你的整个春天✨

姐妹们！春天真的来啦🌸
今天分享5套超好看的春日穿搭...

（后面省略）
```

### 2.3 5个必学提示词技巧

**技巧一：给AI一个角色**

| 差的提示词 | 好的提示词 |
|------------|------------|
| "写一篇销售文案" | "你是一个资深销售文案专家，擅长写高转化率的文案。请写一篇销售文案" |

**为什么有效？**
角色设定让AI知道用什么视角、什么风格来回答。

**技巧二：给具体的要求**

| 差的提示词 | 好的提示词 |
|------------|------------|
| "写一篇小红书" | "写一篇小红书笔记，主题是春日穿搭，字数300字，带5个emoji，风格活泼" |

**为什么有效？**
AI需要明确的方向，越具体越好。

**技巧三：给示例**

| 差的提示词 | 好的提示词 |
|------------|------------|
| "写一个吸引人的标题" | "写一个吸引人的标题，参考这种风格：'5套穿搭承包你的春天！'、'小个子必看！这样穿显高10cm'" |

**为什么有效？**
示例让AI知道你想要什么风格。

**技巧四：让AI分步骤思考**

| 差的提示词 | 好的提示词 |
|------------|------------|
| "写一篇商业计划书" | "请分步骤完成：1. 先分析市场需求；2. 再设计产品方案；3. 最后写出完整计划书" |

**为什么有效？**
分步骤让AI的思考更有条理，输出质量更高。

**技巧五：迭代优化**

```
第一轮：让AI写初稿
第二轮：让AI优化"请让语言更活泼"
第三轮：让AI调整"请再加一些具体案例"
```

**为什么有效？**
AI不是一次就能输出完美的，迭代优化能让结果越来越好。

## 【第三部分：建立自己的提示词库】（约5分钟）

### 3.1 为什么要建立提示词库？

三个原因：

| 原因 | 说明 |
|------|------|
| **提高效率** | 不用每次重新写提示词 |
| **保证质量** | 经过验证的提示词，输出稳定 |
| **快速复制** | 可以把好用的提示词分享给别人 |

### 3.2 如何建立提示词库？

**方法一：用文档记录**

创建一个文档，按场景分类：

```
【提示词库】

一、小红书文案
1. 穿搭类
提示词：...
效果：好评

2. 美食类
提示词：...
效果：待验证

二、销售话术
...

三、商业文案
...
```

**方法二：用Notion/飞书管理**

更专业的方式是用Notion或飞书建立数据库：
- 可以添加标签
- 可以搜索
- 可以持续迭代

## 【第四部分：进阶技巧——让AI成为你的全能助理】（约5分钟）

### 4.1 如何让AI记住你的偏好

**方法：在对话开头设置上下文**

```
我是一个小红书博主，粉丝主要是20-30岁的年轻女性。
我偏好风格：活泼、亲切、带emoji
我常用的称呼：姐妹们、宝子们
```

AI会在整个对话中记住这些偏好。

### 4.2 如何让AI持续优化输出

**技巧：告诉AI哪里不满意，让它改**

```
【第一轮】
AI输出：...（一篇文章）

【第二轮】
你：这段写得太正式了，请改得更口语化一些

AI输出：...（修改后的版本）

【第三轮】
你：很好！但可以再加一个真实案例吗？

AI输出：...（最终版本）
```

**关键**：把AI当作你的助手，不断对话，不断优化。

### 4.3 常见问题解决

**问题一：AI回答太笼统**

解决：让AI具体化
```
差："写一篇文章"
好："写一篇500字的文章，包含3个真实案例，每个案例100字"
```

**问题二：AI回答风格不对**

解决：明确风格要求
```
差："写一段话"
好："写一段话，风格像朋友聊天，不要太正式"
```

**问题三：AI回答太长/太短**

解决：明确字数
```
差："写一段介绍"
好："写一段100字左右的介绍"
```
